# 平安证券体验报告静态页面实施计划

## 需求概述

将 `平安证券预发布版本体验报告.md` 转换为一个纯前端静态页面，使用 HTML + CSS + JavaScript + Tailwind CSS，页面风格参考平安证券官方网站。

## 技术选型

| 技术 | 选型 | 说明 |
|------|------|------|
| 核心语言 | HTML5 + CSS3 + JavaScript (ES6+) | 纯前端，无依赖框架 |
| CSS框架 | Tailwind CSS v3.x | CDN引入，快速开发 |
| 图表库 | Chart.js | CDN引入，用于数据可视化 |
| 图片预览 | lightbox2 或原生实现 | 简单的图片预览功能 |

## 页面设计规范

### 配色方案（参考平安证券官网）

| 颜色 | 色值 | 用途 |
|------|------|------|
| 品牌蓝 | #1E88E5 | 主色调、导航栏、主要按钮 |
| 平安橙 | #FF6B00 | 高亮色、强调文字、收益率 |
| 深灰 | #333333 | 正文文字 |
| 中灰 | #666666 | 辅助文字 |
| 浅灰 | #F5F7FA | 背景色、卡片背景 |
| 边框灰 | #E0E0E0 | 分割线、边框 |
| 严重红 | #D32F2F | P0问题标识 |
| 重要橙 | #F57C00 | P1问题标识 |
| 一般蓝 | #1976D2 | P2问题标识 |
| 轻微绿 | #388E3C | P3问题标识 |

### 字体规范

- **字体族**: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif
- **标题**: 加粗，字号从大到小递减
- **正文**: 常规字重，14-16px
- **数字**: 使用等宽字体展示数据

## 风险与注意事项

- **图片路径**: 确保所有图片使用相对路径正确引用
- **CDN依赖**: 所有外部依赖使用稳定的CDN链接
- **浏览器兼容性**: 确保在主流浏览器（Chrome、Firefox、Safari、Edge）中正常工作
- **性能优化**: 图片可考虑添加懒加载
